Abaqus 接触

![](/Abaqus contact/con1.jpg)

Pressure-Overclosure

Hard Contact

![](/Abaqus contact/con2.png)

  • 间隙为0时，开始接触
  • 对接触压力传递的幅值无限制
  • 无穿透

Exponential

![](/Abaqus contact/con3.jpg)

  • $0<c<c_0$之间接触压力呈指数上涨，其中间隙$c_0$、接触压力$P_0$和接触刚度上限（Abaqus/Explicit, optional）$k_{max}$需要定义。
  • 默认情况下，对于kinematic contact method，$k_{max}$为无穷大
  • 对于penalty contact method，$k_{max}$为默认罚刚度。
  • 罚函数接触算法中，刚度过大会导致稳定时间增量的缩减，引入$k_{max}$可以有效改善这种不利影响。

Linear

Tabular form中的一种特殊形式，过原点，定义接触刚度$h>0$

Tabular

![](/Abaqus contact/con4.png)

  • 以表格形式分段定义接触压力-过盈接触关系，其中间$h＜0$阶段对应过盈接触。
  • 每相邻两点之间为线性关系，曲线整体走势为正相关；
  • 对于general contact in abaqus/explicit 0；外插。

Scale Factor (General Contact, Explicit)

![](/Abaqus contact/con5.png)

  • 另一种线性分段表征接触压力-过盈接触的方法，各阶段过盈量以最小单元尺寸为标准，通过引入geometric scale factor(contact stiffness scale factor)强化默认接触刚度
  • 以第i阶段为例，，需要定义overclosure factor(or measurement), contact stiffness scale factor 和initial stiffness scale factor

Constraint enforcement method

![](/Abaqus contact/con6.png)

default

具体实施方法取决于具体的相互作用

direct

  • 严格执行定义的pressure-overclosure behavior，无近似替代或增广迭代，因而在求解过程中容易产生过约束问题(主从接触对也有类似局限性)；
  • 可严格执行硬接触；
  • 唯一可以执行软接触行为的方式；
  • 不适用于three-dimensional self-contact + node-to-surface类型;某些二阶面单元在硬接触中表现一般。

penalty

![](/Abaqus contact/con7.png)
Hard pressure-overclosure relationship的近似处理，允许一定程度的穿透，接触应力与穿透距离成正比。可有效避免过约束问题。

Linear

  • Penalty stiffness 默认为接触区域单元刚度的10倍或者自定义；
  • Stiffness scale factor 比例系数，和罚刚度相乘；
  • Clearance…..zero 接触压力为零时的间隙，默认为0。

NonLinear

![](/Abaqus contact/con8.jpg)
Pressure-overclosure relationship分为四个阶段，接触刚度分别为0、常数、线性增加、常数。

  • 无接触阶段：间隙大于$c_0$，接触压力为零
  • 第一常数阶段：过盈接触为$(-c_0,e)$，接触刚度为常量插$K_i$，接触压力呈线性增长，接触刚度取值于对应接触区代表性单元的刚度，$e$默认取值为面元特征长度的1%
  • 加强阶段：过盈接触为$(e,d)$，接触刚度线性增加，下限为插$K_i$，上限$K_f$取值于对应接触区代表性单元的刚度的100倍，$d$$默认取值为面元特征长度的3%
  • 第二常量阶段：过盈接触大于$d$，接触刚度为常量$K_f$

对于罚接触而言，初始刚度取较小值有较好的收敛效果和鲁棒性，对于第二常量阶段，随着接触压力的增加，穿透控制在较为合理的范围内。对于线性摄动分析，接触刚度取值大小为$100K_f$

  • Maximum stiffness value 默认或自定义；
  • Stiffness scale factor 与线性相同；
  • Initial/final stiffness ratio 插19.png ；
  • Upper quadratic limit scale factor 面元特征尺寸比例系数；
  • Lower quadratic limit ratio 插20.png；
  • Clearance … zero 同线性。

Augmented Lagrange

在penalty method的基础上增强迭代以减少穿透距离，只适用于硬接触，穿透容许可自定义(只能添加关键字修改)

To specify an absolute penetration tolerance:
*CONTACT CONTROLS, ABSOLUTE PENETRATION TOLERANCE=tolerance
To specify a relative penetration tolerance:
*CONTACT CONTROLS, RELATIVE PENETRATION TOLERANCE=tolerance

Allow separation after contact

在显式动力学分析中，适用于通用接触+主从接触对，不能应用于adaptive meshing；不分离选项常用于rough friction model，既不分离也没有切向滑动。
